# eslint, prettier

```
npm install --save-dev eslint eslint-plugin-jest
```

# 기능 구현 목록

- 입력

  - [] 생성할 다리 길이 입력 받기
    - 3 이상 20 이하
    - `이동할 칸을 선택해주세요. (위: U, 아래: D)`
  - [] 이동할 칸을 입력 받기
    - U와 D만 입력가능

- 출력

  - [] 게임 시작 문구 `다리 건너기 게임을 시작합니다.`
  - [] 최종 게임 결과
  - [] 현재 결과

- 비지니스

  - [x] 입력받은 값만큼 다리 생성
  - [] 게임이 끝났는지 확인
  - [] 이동할 수 있는지 확인
  - [] 시도한 횟수 저장
  - [] 0인경우 D, 1인경우 U

- 예외처리
  - [] 잘못된 다리 길이 입력
    - `[ERROR] 다리 길이는 3부터 20 사이의 숫자여야 합니다.`
  - [] U, D 이외의 값
